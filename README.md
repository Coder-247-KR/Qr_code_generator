# 🔗 QR Code Generator

A simple and modern QR Code Generator built with HTML, CSS, and JavaScript. Instantly generate QR codes for any website URL and scan them using any smartphone or QR code scanner.

## 🚀 Live Demo

👉 https://qr-generator-murex-pi.vercel.app/

## ✨ Features

- Generate QR codes from any URL
- Instant QR code generation
- Press **Enter** or click the button to generate
- Clean and responsive design
- Mobile-friendly interface
- Fast and lightweight

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- QRCode.js

## 📖 How to Use

1. Enter a website URL.
2. Press **Enter** or click the **Generate QR Code** button.
3. Scan the generated QR code using your smartphone.
